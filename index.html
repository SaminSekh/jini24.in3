🗄️ Google Sheet Database Setup

Connect your live Google Sheet to power the entire platform

📊 This will create 4 sheets in your Google Sheet:
Services
ServiceID, Name, Category, Price, Image, Description, Status
Bookings
BookingID, Customer, Service, Date, Status, Partner, Price, Payment…
Partners
PartnerID, Name, Phone, Skills, Areas, Rating, Status
Payment_Settings
PaymentMethodID, Name, Details, QRCodeURL, Status
1
Copy the Apps Script Code

Click the button below to copy the complete, ready-to-use backend code to your clipboard.

2
Create Google Sheet & Open Apps Script
  1. Go to sheets.google.com and create a new blank spreadsheet.
  2. Click Extensions in the top menu → Apps Script.
  3. In the editor that opens, delete all existing code in Code.gs.
  4. Paste the code you copied in Step 1 (Ctrl+V).
  5. Click the Save icon (💾) or press Ctrl+S.
3
Initialize the Sheets (Run Setup)
  1. In the Apps Script editor, click the function dropdown (top bar) and select setupSheets.
  2. Click the ▶ Run button.
  3. When prompted, click Review Permissions → choose your Google account → click Allow.
  4. A popup will confirm: "✅ Setup complete! All sheets initialized."
  5. Go back to your spreadsheet — you will see 4 sheets created: Services, Bookings, Partners, Payment_Settings.
⚠️ Already set up before? Manually add these columns to your Bookings sheet if missing: PaymentStatus · PaymentMethodUsed · TransactionRef. Also create a Payment_Settings sheet with columns: PaymentMethodID · Name · Details · QRCodeURL · Status.
4
Deploy as Web App
  1. In Apps Script, click Deploy (top right) → New deployment.
  2. Click the ⚙️ gear icon next to "Select type" → choose Web app.
  3. Set Execute as: Me and Who has access: Anyone.
  4. Click Deploy → authorize if prompted.
  5. Copy the Web App URL that appears (starts with https://script.google.com/macros/s/...).
💡 Re-deploying? Go to Deploy → Manage deployments → ✏️ Edit → New version → Deploy to push code changes without changing your URL.
5
Paste Your Web App URL & Connect

Saved locally in your browser. Connects this app to your live Google Sheet database.
